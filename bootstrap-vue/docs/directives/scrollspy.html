Scrollspy 滚动监听

根据滚动位置自动更新Bootstrap导航或列表组组件，用于指示可视窗口视口中当前处于活动状态的链接。

运行原理

v-b-scrollspy 指令对正常运行有一些要求：

  • 必须将其应用于包含 <b-nav> 或 <b-list-group> 组件的元素/组件，您要在其中指示当前处于活动状态的链接。
  • Scrollspy 需要 位置: 相对;相对 在您监视的元素上，通常是 <body>.
  • 监视除 <body>以外的元素时, 请确保已 设置 高度并 溢出-y: scroll; 应用.
  • 锚点 (<a>, <b-nav-item>, <b-dropdown-item>, <b-list-group-item>) 是必需的，并且必须具有指向的 href (通过 href 或 props) 到您正在监视的容器中具有该 id 的元素. 使用 prop时, 要么将 路径 设置为已 #id-of-element结尾, 要么将位置属性hash 设置为 #id-of-element.

成功实施后, 您的导航或列表组将进行相应的更新, 根据其关联的 目标 将活动状态从一项移至下一项.

在带有 prop在组件上使用scrollspy

当使用 Vue Router (或 Nuxt.js) 时, 并且您正在使用 to prop生成链接时,请使用以下方法之一在渲染的链接上生成适当的 href： 

<!-- using a string path -->
<b-nav-item to="#id-of-element">link text</b-nav-item>

<!-- using a router `to` location object -->
<b-nav-item :to="{ hash: '#id-of-element' }">link text</b-nav-item>

Scrollspy 可同时使用 历史记录 和 哈希路由模式 ，只要生成的URL以 #id-of-element结尾即可.

指令语法和用法

v-b-scrollspy:arg.mod1.mod2="option"

哪里:

  • arg 是要监视滚动的元素的 ID (减去 #) .可选 (默认为 body. 可以被选项 覆盖)
  • mod1 和 mod2 可以是移偏号或字符串 方法 (请参见下面的配置对象). 修饰符的顺序并不重要. 两者都是可选的
  • option 可以是标识要监视 滚动的 元素的字符串, 数字偏移量或配置对象(请参见下文). 可选在

注意: 与本机 Bootstrap v4 相比. 在 BootstrapVue中 将 v-b-scrollspy 伪指令应用于具有要激活的链接的目标元素, 而arg 或 option 指定要监视 (间谍)滚动在元素.

指令a可以应用于任何包含 <nav-item>, <b-dropdown-item>, <b-list-group-item> (或带有适当类的 <a> 标签), 的包含元素或组件 因为它们已经呈现了 href 属性，该属性指向滚动元素中具有相应 id的元素.

配置对象属性

const config = {
  element: 'body',
  offset: 10,
  method: 'auto',
  throttle: 100
}
属性 类型 默认 描述
element 字符串或引用 'body' 要监视滚动的元素. 可以是 ID (#foo), css 选择器 (#foo div), 或对元素/组件节点的引用.如果是 CSS 字符串, 则使用第一个匹配元素. 如果使用 ID 则必须以 #开头.
offset 数 10 在触发活动状态之前，从滚动视口顶部开始的偏移量（以像素为单位）.
method 串 'auto' position 将计算相对于滚动容器的目标偏移量. offset 将计算相对于窗口/视口顶部的目标偏移量. 如果 滚动 元素为 body, 则auto将选择offset 否则为position.
throttle 数 100 调整大小 事件 在重新计算偏移之前停止触发的超时（以毫秒为单位）.

如果参数/修饰符和值（对象或数字）被传递，则该值优先于参数arg和修饰符.

如果任何选项的类型无效，那么会将错误写入控制台.

配置说明

  • 如果没有scroll元素，则假定在<body>上滚动
  • 如果scroll元素是CSS选择器，则选择第一个找到的元素
  • 如果找不到滚动元素，则ScrollSpy会静默不执行任何操作

重要! 需要相对定位 无论实现方法如何, scrollspy 都需要使用 position: relative; 在要滚动的元素上.在大多数情况下，这是 <body>.在 <body>以外的元素上滚动查看时, 请确保已设置 CSS 高度 并 溢出-y: scroll; 应用.

事件用例

每当激活目标时，都会在$ root上发出 bv:scrollspy::activate 事件 并以目标在 ID 作为参数 (即 #bar)

const app = new Vue({
  el: '#app',
  created() {
    this.$root.$on('bv::scrollspy::activate', this.onActivate)
  },
  methods: {
    onActivate(target) {
      console.log('Received event: "bv::scrollspy::activate" for target ', target)
    }
  }
})
