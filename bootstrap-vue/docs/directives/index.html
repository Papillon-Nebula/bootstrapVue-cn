指令目录

BootstrapVue的指令和指令插件。

Hover 状态指令 — 的轻型指令，允许您在元素悬停或未悬停时进行反应 v2.5.0 Popover 提示备注 — 使用Bootstrap v4 CSS进行样式和动画处理，将BootstrapVue popover 提示备注效果 添加到站点上的任何元素。 弹出窗口是类固醇的工具提示。 Scrollspy 滚动监听 — 自动根据鼠标滑轮滚动位置激活BootstrapVue导航或列出组组件，以指示当前在视口中处于活动状态的链接。 Tooltip提示指令 - 将自定义BootstrapVue工具 提示 添加到任何元素。 可以通过将鼠标悬停，聚焦或单击某个元素来触发工具提示。 Visible 可见性操作 - 当一个元素在视口中可见时，`vb-visible`指令允许您做出反应。 v2.1.0
